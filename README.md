# Aura Zed Extension

Syntax highlighting support for the [Aura](https://github.com/nahharris/aura) programming language in [Zed](https://zed.dev).

## Installation

### From Zed

1. Open Zed
2. Open the Extensions panel (Ctrl+Shift+X or Cmd+Shift+X)
3. Search for "Aura"
4. Click Install

### Dev Installation

1. Clone this repository
2. In Zed, open the Extensions panel
3. Click "Install Dev Extension"
4. Select the `aura-zed` directory

## Features

- Syntax highlighting
- Bracket matching
- Auto-indentation
- Code outline (functions, definitions, macros)
